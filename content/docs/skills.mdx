# Skills library

Claude Code slash commands that run the hypothesis validation loop — from writing a hypothesis contract to closing an experiment with evidence. Each skill is a `.md` file in `.claude/skills/` in your repo.

Install the core workflow:

```sh
$ npx systemix workflow add hypothesis-validation
```

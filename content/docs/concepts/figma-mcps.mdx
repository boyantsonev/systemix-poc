# Figma MCPs

Systemix uses two separate Figma MCP servers. They are complementary, not alternatives — one reads, one writes. Understanding which to call when is the single most important thing to know before running any Figma skill.

---

## The two servers

|  | Official Figma MCP | Figma Console MCP |
|---|---|---|
| Made by | Figma Inc. | Southleft / TJ Pitre |
| Tool prefix | `mcp__claude_ai_Figma__*` | `mcp__figma-console__*` |
| Tools | 16 (1 generic write: `use_figma`) | 94 (35+ dedicated writes) |
| Connection | REST API + Figma cloud | WebSocket Desktop Bridge + REST |
| Auth | OAuth | Personal Access Token |
| Source | Closed | Open (MIT) |
| Cost | Usage-based → paid | Free, unlimited, self-hostable |
| Requires | Figma OAuth (one-time login) | Figma Desktop open on port 3845 |

- [figma.com/developers/mcp](https://www.figma.com/developers/mcp)
- [docs.figma-console-mcp.southleft.com](https://docs.figma-console-mcp.southleft.com)

---

## Decision rule

One sentence: **read with Official, write with Console.**

**Reading from Figma → Official**

Use `mcp__claude_ai_Figma__*` for all reads: `get_design_context`, `get_variable_defs`, `get_screenshot`, `get_metadata`. No Figma Desktop required.

**Writing to Figma → Console**

Use `mcp__figma-console__*` for all writes: creating/updating variables, setting fills, pushing images, renaming nodes. Requires Figma Desktop open.

**Code Connect → Official only**

Code Connect mapping (`get_code_connect_map`, `send_code_connect_mappings`) exists only in the Official MCP. Console has no equivalent.

**Batch variable ops → Console only**

`figma_batch_create_variables` and `figma_batch_update_variables` process 100 entries per call. The Official MCP has no batch path — each variable would be a sequential call.

---

## Exclusive capabilities

**Only Official MCP can:**
- Code Connect — map Figma components to codebase React components
- Framework code generation (React, Vue, etc.) from a Figma node
- Capture any live URL as a Figma layer via `use_figma`
- Create FigJam diagrams from Mermaid syntax
- Create new blank Figma files

**Only Console MCP can:**
- Batch variable create/update (100 per call)
- Design system health scoring and hardcoded value detection
- Real-time selection monitoring and document change events
- `figma_execute` — run arbitrary Plugin API code
- FigJam (9 tools), Slides (15 tools), Annotations (3 tools)

## If Console MCP errors

Console MCP write tools fail with `bridge not connected` or `no open file` when Figma Desktop is not running or the target file is not open. Fix in order:

1. Open Figma Desktop and open the target file
2. Reload the Figma Console MCP plugin inside Figma (Plugins → Development → Figma Console MCP → Run)
3. Verify the MCP server is registered: `claude mcp list` should show `figma-console`
